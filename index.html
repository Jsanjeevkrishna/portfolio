Who I Am

I'm Sanjeev Krishna, a passionate full-stack developer with a hunger for building products that solve real problems. I love working across the entire spectrum — from crafting pixel-perfect React UIs to engineering robust Node.js backends, and from integrating LLMs into production apps to building physics-driven Unity 3D games.

My most ambitious project, CodeSync, is a real-time collaborative code editor (think Google Docs for code) — deployed live on Vercel & Render with Socket.IO, Monaco Editor, and multi-language execution support. DR-AI brings together a MERN stack and DeepSeek AI to create an intelligent medical consultation system with 3D anatomical interaction.

On the game dev side, I build immersive Unity experiences — from a horror first-person shooter to a 3D drone vs enemy combat simulation with NavMesh AI and 6-DOF flight physics.
